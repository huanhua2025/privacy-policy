## Overview

Duplicate Media Cleanup (hereinafter referred to as "this application") is a local macOS desktop utility designed to help users identify and remove duplicate image and video files. **This application does not collect, store, transmit, or share any personal information.**

## Information Collection and Use

**This application does not collect any information.** Specifically:

-   **No network access**: The application runs entirely locally and does not connect to any remote server, API, or third-party service.
-   **No user accounts**: No registration, login, or account creation is required.
-   **No analytics**: No analytics, crash reporting, or telemetry SDKs are used.
-   **No advertising**: The application contains no ads or third-party tracking code.

## File Access

-   Users manually select a directory to scan via the system file picker (NSOpenPanel).
-   The application only accesses files within the authorized directory after explicit user consent.
-   The application **does not** access files outside the authorized directory.
-   The selected directory path is temporarily held in memory only during the application session and is not persisted when closed.
-   Deletion operations move files to the Trash via macOS system API; files are not permanently erased.

## Data Storage

-   The application does not use a local database or any persistent storage to save user data.
-   Scan results are kept only in memory and are cleared when the application is closed.
-   User preferences (language and theme selection) are stored in the system standard UserDefaults, containing only language codes and theme options — no file paths or file content.

## Third-Party Services

This application does not integrate any third-party services, SDKs, or APIs.

## Children's Privacy

This application does not collect information from users of any age, including children under 13.
